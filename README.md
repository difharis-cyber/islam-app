# ☪ تعلّم الإسلام — App Islam

Guide d'apprentissage personnel de l'Islam.

## Contenu

- 📖 **Cours** — Fondements, Pratique, Approfondissement
- 🕌 **Sourates** — 15 sourates avec arabe, phonétique et traduction
- ✨ **99 Noms d'Allah** — Avec explications détaillées
- 📿 **Dhikr** — 14 formules avec bienfaits
- 🤲 **Doua** — Invocations du Coran, de la Sunna et des Prophètes

## Stack

- React 18
- Vite
- Déployé sur Vercel

## Lancer en local

```bash
npm install
npm run dev
```
